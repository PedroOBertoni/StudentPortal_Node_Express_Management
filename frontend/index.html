Bem-vindo ao Sistema de Alunos

Este projeto foi desenvolvido pelos alunos Pedro de Oliveira Bertoni e João Pedro Lourenço Ferreira como parte das atividades da disciplina Estrutura de Dados II, ministrada pelo professor Alexandre.

Atualmente cursamos o 4º semestre de Engenharia de Software na Uni-FACEF. O objetivo deste trabalho é consolidar conceitos estudados em sala por meio de uma aplicação prática que auxilie no gerenciamento e geração de relatórios de alunos.

A solução implementa algoritmos fundamentais de estrutura de dados e ordenação — em específico, Busca Sequencial para pesquisa de registros e Selection Sort para ordenação de listas — permitindo:

Recursos do Sistema

  • Geração de relatórios organizados e filtráveis;
  • Ordenação dos dados por diferentes campos (nome, RA, idade, média);
  • Filtragem para exibir apenas alunos aprovados;
  • Interface simples e responsiva para fácil uso por professores e alunos.

Este projeto serve tanto como exercício acadêmico quanto como base inicial para futuras melhorias — por exemplo, substituição de algoritmos por versões mais eficientes, persistência em banco de dados e funcionalidades de exportação (CSV/PDF).
